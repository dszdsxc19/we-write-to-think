# 媒体与设计工具

## 1. 视频工具

### 播放

**IINA** (已安装)

免费开源，倍速播放、在线字幕、画中画。

### 录制

**OBS** (已安装)

免费开源直播录制软件，推流、录屏、虚拟摄像头。

### 转码

**HandBrake**

```bash
brew install --cask handbrake
```

免费开源视频转码，支持几乎所有格式。

### 下载

**Downie**

```bash
brew install --cask downie
```

视频下载工具，支持 1000+ 网站。

---

## 2. 图片工具

### 查看

**Xnip** (推荐)

滚动截图、贴图、标注。

**CleanShot X** (付费)

截图增强，录制 GIF、标注功能丰富。

### 编辑

**Pixea**

```bash
brew install --cask pixea
```

免费图片查看器，EXIF 信息显示。

**Photomator**

AI 修图，批量处理。

---

## 3. 设计工具

### UI 设计

**Figma**

```bash
brew install --cask figma
```

在线 UI 设计工具，协作友好。

### 截图标注

**Easydiff**

图片标注，箭头、框选、文字。

### 配色

**ColorSlurp**

取色工具，色板管理。
