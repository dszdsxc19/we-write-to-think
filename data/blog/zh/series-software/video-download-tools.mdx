# 视频下载工具指南

## 1. Downie - 主流选择

> 付费但强大，支持 1000+ 网站

```bash
brew install --cask downie
```

### 支持平台

| 平台        | 说明           |
| ----------- | -------------- |
| YouTube     | 4K、8K、字幕   |
| Bilibili    | 弹幕、字幕、4K |
| 抖音/TikTok | 视频下载       |
| Instagram   | 视频和图片     |
| Twitter/X   | 视频           |
| Vimeo       | 高画质         |
| 微博        | 视频           |
| 爱奇艺      | 部分支持       |
| 腾讯视频    | 部分支持       |

### 使用方法

1. 复制视频链接
2. 打开 Downie
3. 自动检测并下载
4. 可选择格式和画质

### 推荐设置

- **默认格式**: MP4
- **默认画质**: 最高可用
- **保存位置**: ~/Downloads/Videos
- **子文件夹**: 按平台名称分类

## 2. yt-dlp - 命令行方案

> 免费开源，功能强大

```bash
brew install yt-dlp ffmpeg
```

### 基本用法

```bash
# 下载 YouTube 视频
yt-dlp "https://www.youtube.com/watch?v=VIDEO_ID"

# 下载 B站视频
yt-dlp "https://www.bilibili.com/video/BV1xxx"

# 下载 4K 画质
yt-dlp -f "bestvideo[ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]" "URL"

# 下载并转换音频
yt-dlp -x --audio-format mp3 "URL"

# 下载字幕
yt-dlp --write-subs --sub-lang zh "URL"
```

### 常用参数

| 参数           | 说明           |
| -------------- | -------------- |
| `-f`           | 选择格式       |
| `-o`           | 输出文件名     |
| `--write-subs` | 下载字幕       |
| `-x`           | 提取音频       |
| `--playlist`   | 下载播放列表   |
| `-a`           | 从文件读取链接 |

### 配置文件

```bash
# ~/.config/yt-dlp/config
-f "bestvideo[ext=mp4]+bestaudio[ext=m4a]/best[ext=mp4]/best"
-o "~/Downloads/Videos/%(title)s.%(ext)s"
--sub-langs "zh,en"
--embed-subs
```

## 5. 注意事项

### 版权和合规

- 仅下载有权下载的内容
- 遵守平台使用条款
- 用于个人学习研究

### 下载质量

- B站支持 4K、杜比音效
- YouTube 支持 8K、VR
- 抖音视频有水印

### 字幕处理

```bash
# 下载视频和字幕
yt-dlp --write-subs --sub-lang zh --embed-subs "URL"

# 下载 ASS 弹幕 (B站)
yt-dlp --write-subs --sub-format ass "URL"
```
