# macOS 常用软件指南

> 本文持续更新，推荐安装顺序。

## 工具哲学

**约定大于配置，现代化，速度快**

选择工具的标准：

- 默认行为符合直觉，无需大量配置
- 积极维护，社区活跃
- 性能优先，原生应用优先
- 隐私保护

---

## 1. 系统工具

### 解压

```bash
brew install --cask the-unarchiver
```

免费开源的解压工具，支持 zip, 7z, rar 等格式。

### 系统清理 - Mole

```bash
brew install tw93/tap/mole
```

开源系统清理工具，由国内开发者出品。

功能：

- 系统垃圾清理
- 应用缓存清理
- 隐私数据清除
- 轻量快速

---

## 2. 效率工具

### 启动器 - Raycast

```bash
brew install --cask raycast
```

macOS 启动器，Alfred 的现代替代品。

**推荐扩展**：

- Window Management: 窗口管理
- Clipboard History: 剪贴板历史
- Color Picker: 取色器
- Emoji Search: 表情符号
- GitHub: 仓库搜索

### 截图工具 - Shottr

```bash
brew install --cask shottr
```

功能全面的截图工具，支持滚动截图、标注、贴图。

---

## 3. 媒体工具

### 视频播放

```bash
brew install --cask iina
```

免费开源的 macOS 视频播放器，简洁美观，倍速播放。

### 视频下载

```bash
brew install --cask downie
```

视频下载工具，支持 1000+ 网站（B站、YouTube、抖音等）。

### 视频转码

```bash
brew install --cask handbrake
```

免费开源的视频转码工具，支持几乎所有格式。

### 直播录制 - OBS

```bash
brew install --cask obs
```

免费开源直播录制软件，推流、录屏、虚拟摄像头。

### PDF 阅读

```bash
brew install --cask skim
```

免费 PDF 阅读器，专注阅读体验。
